# Analysis Notebooks

Jupyter notebooks for exploratory data analysis and visualisation.

## Notebook Index

| Notebook | Description |
|----------|-------------|
| `01_sensor_calibration.ipynb` | Characterise sensor noise floor and apply calibration corrections |
| `02_survey_transect.ipynb` | Visualise a survey transect overlaid on a gravity anomaly heatmap |
| `03_grace_comparison.ipynb` | Compare GADV measurements against GRACE satellite free-air anomaly grids |
| `04_noise_analysis.ipynb` | Allan deviation and PSD analysis of accelerometer data |

## Running Notebooks

```bash
cd analysis/notebooks
jupyter notebook
```
