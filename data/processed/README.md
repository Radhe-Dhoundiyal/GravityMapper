# Processed Data

Cleaned and calibrated data files produced by `analysis/scripts/preprocess.py`.

Files here are derived from raw exports and are safe to use as analysis inputs. Each processed file should have a corresponding entry in `experiments/logs/` describing the processing parameters applied.

Processed files use the same naming convention as raw files with a `_processed` suffix:

```
YYYY-MM-DD_<location>_<session-id>_processed.csv
```
