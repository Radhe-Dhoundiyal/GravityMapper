# Circuit

Electrical schematics and PCB design files for the ESP32-based sensor node.

## Contents

- Wiring diagrams for sensor connections to the ESP32 development board
- Power regulation circuit (LiPo battery → 3.3 V / 5 V rails)
- Signal conditioning for analog sensor outputs
- Antenna placement recommendations for GPS reliability

## Design Tools

Schematics are authored in KiCad. Export Gerber files to this directory before fabrication.
