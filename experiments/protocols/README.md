# Measurement Protocols

Standardised field protocols for GADV surveys.

## Protocol Index

| Protocol | Description |
|----------|-------------|
| `P-001_site-selection.md` | Criteria for selecting survey sites and establishing control points |
| `P-002_pre-survey-checklist.md` | Equipment checks, sensor warm-up, and software startup procedure |
| `P-003_transect-survey.md` | Step-by-step procedure for conducting a linear transect survey |
| `P-004_grid-survey.md` | Procedure for a regular grid survey over a target area |
| `P-005_data-export.md` | Exporting, labelling, and archiving data after a survey session |

Protocols should be reviewed and updated after each experiment based on lessons learned.
