# Experiments

Documentation for field experiments conducted with the GADV platform.

## Subdirectories

### protocols/
Standardised measurement protocols describing how to set up the rover, configure the software, and conduct a repeatable survey. Each protocol is a markdown file that can be printed and used in the field.

### logs/
Field logs for completed experiments. Each log records the date, location, weather conditions, equipment configuration, any anomalies observed, and links to the corresponding raw data files in `data/raw/`.

## Experiment Naming

Experiments are numbered sequentially: `EXP-001`, `EXP-002`, etc. Protocol and log files reference this identifier.
