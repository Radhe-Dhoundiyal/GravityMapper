# Experiment Logs

Field logs for completed GADV experiments.

## Log Format

Each log is a markdown file named `EXP-NNN_YYYY-MM-DD_<location>.md` and contains:

- **Date & Location**: GPS coordinates and site description
- **Personnel**: Names of operators present
- **Equipment**: Sensor node firmware version, rover hardware revision
- **Weather**: Temperature, humidity, wind speed, sky conditions
- **Procedure**: Protocol reference and any deviations
- **Observations**: Notable events, sensor anomalies, or environmental factors
- **Data Files**: Links to raw data files in `data/raw/`
- **Status**: Complete / Partial / Aborted

## Example

```
EXP-001_2025-04-01_campus-north.md
```
